# Dragon Ball Z API - APP ANDROID

App Android que consume una API de Dragon Ball para mostrar personajes, planetas y poderes.

## Tecnologías
- Kotlin
- Retrofit
- MVVM
- Clean Architecture
- Jetpack Compose
- Coil

## Funcionalidad
- Lista de personajes
- Consumo de API
- Manejo de estados (cargando, éxito, error)
- Carga de imágenes

## Cómo ejecutar
1. Clonar el repo
2. Abrir en Android Studio
3. Ejecutar
